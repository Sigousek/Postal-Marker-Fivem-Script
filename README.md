# Postal-Marker-Fivem-Script
Optimized simple marker script. example:"/postal 8022"  it gets marker on the map


How to use:
1. Download from code Downlad zip
2. Put it in your "resource" folder
3. put "ensure postalmarker" in your server.cfg
4. Have Fun
